주제: GPT를 활용한 단어암기 어플리케이션

- 개요:
  - 원하는 단어만으로 단어장을 만들어서 공부할 수 있다.
  - 단어 암기가 쉽도록 GPT가 단어 문제를 만들어준다.

- 목표:
  - 개발 계획
  - 시간계획

- 기능 구성
  - 태그 기반 단어 입력 기능
  - 태그 기반으로 단어를 묶고, GPT 기반 단어 문제 제작 기능

- 메뉴 구성
  - 홈
  - 입력한 단어 목록보기
  - 퀴즈만들기(테스트 구성하기)
  - 랜덤구성
  - 태그를 선택해 테스트 구성
  - 오답노트
  - 즐겨찾기 보기
